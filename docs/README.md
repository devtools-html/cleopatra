# perf.html development docs

This project is a client for reading profiles from the Gecko Profiler and potentially other sources in the future. It is written using JavaScript and [Flow types](https://flow.org/) using [React](https://facebook.github.io/react/) for the UI components and [Redux](http://redux.js.org/) for the state management.
